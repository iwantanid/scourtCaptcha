# 대법원 나의 사건 검색 캡차 디코더 (Scourt Captcha Decoder)

NVIDIA CUDA GPU 가속을 활용하여 대법원 나의 사건 검색 사이트의 6자리 숫자 캡차(CAPTCHA)를 판독하는 PyTorch 기반 딥러닝 디코더 엔진입니다. 

본 프로젝트는 인공지능 에이전트와의 페어 프로그래밍 및 바이브 코딩(Vibe Coding) 기법을 활용하여 데이터 수집부터 정제, CNN 모델 설계, 웹 API 서빙까지 머신러닝 파이프라인의 전 과정을 구축하는 데 목적을 두고 개발되었습니다.

---

## 주요 성과 및 특징

*   **높은 판독 정확도**: 검증 데이터셋 기준 6자리 숫자 완전 일치 정답률 **99.34%** 달성 (글자 단위 정확도 99.89%).
*   **초고속 추론 속도**: 이미지 1장 판독 시 CPU 환경 약 5ms, GPU 환경 1ms 이내로 응답합니다.
*   **Data-Centric 최적화**: 모델 구조 변경 대신 6,090장의 라벨 데이터셋에서 정답 오기입(Label Noise)을 찾아내고 교정함으로써 성능 한계를 돌파했습니다.
*   **독립적인 모듈 설계**: 관심사 분리(Separation of Concerns) 원칙을 적용하여 수집, 훈련, 배포 모듈을 완벽하게 분리했습니다.

## 프로젝트 모듈 구성

이 저장소는 역할에 따라 크게 3개의 모듈로 나뉩니다. 각 모듈의 상세한 사용법, 설치 방법 및 API 명세는 하위 폴더의 README 문서를 참조하십시오.

### 1. [캡차 데이터 수집 모듈 (scraper)](./scraper/README.md)
*   **기술 스택**: Node.js, Playwright
*   **설명**: 대법원 웹사이트에 접근하여 iframe 내부에 동적으로 렌더링되는 캡차 이미지를 우회 추출하여 대량 수집합니다.
*   **상세 안내**: [`scraper/README.md` 바로가기](./scraper/README.md)

### 2. [모델 훈련 및 검증 모듈 (training)](./training/README.md)
*   **기술 스택**: Python, PyTorch, CUDA
*   **설명**: PyTorch 데이터로더를 통한 실시간 데이터 증강(Augmentation), 합성곱 신경망(CNN) 학습, 검증 데이터 기반 정확도 평가 및 모델 조기 종료(Early Stopping)를 수행합니다. 6,090장의 캡차 원본 데이터셋과 정답지가 포함되어 있습니다.
*   **상세 안내**: [`training/README.md` 바로가기](./training/README.md)

### 3. [웹 서빙 및 API API 모듈 (api)](./api/README.md)
*   **기술 스택**: Python, FastAPI, Uvicorn
*   **설명**: 학습이 완료된 모델 가중치(`.pth`)를 로드하여 RESTful API(`/predict`)를 제공합니다. 또한 브라우저에서 드래그 앤 드롭으로 캡차를 쉽게 판독해볼 수 있는 직관적인 웹 기반 검증 패널을 포함합니다.
*   **상세 안내**: [`api/README.md` 바로가기](./api/README.md)
